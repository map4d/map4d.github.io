# Map4D SDK document

Documentation website được tạo bằng [docsify](https://docsify.js.org/)

## Setup

```bash
npm i docsify-cli -g
```

## Preview

Tại thư mục gốc của map4d.github.io, chạy lệnh sau:

```bash
docsify serve .
```
